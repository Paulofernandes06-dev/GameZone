Sobre o GameZone

O GameZone é um site dedicado ao universo dos jogos competitivos online. Nosso objetivo é reunir em um só lugar informações, notícias, rankings e dicas sobre os jogos mais populares do cenário gamer atual.

Criado como projeto acadêmico da disciplina de Tecnologias para Internet da Uniube, o site busca aplicar na prática os conceitos de desenvolvimento web com HTML e CSS.

Nossa Missão

Oferecer conteúdo de qualidade para jogadores de todos os níveis, desde iniciantes até veteranos do cenário competitivo. Abaixo estão os pilares do GameZone:

  • Notícias atualizadas sobre patches e eventos
  • Dicas e estratégias para melhorar o desempenho
  • Rankings dos melhores jogadores e equipes
  • Galeria de imagens dos jogos em destaque
  • Links para os sites e canais oficiais dos jogos

Como o Site Foi Desenvolvido

O desenvolvimento seguiu as seguintes etapas, em ordem:

  1. Definição do tema e pesquisa de conteúdo
  2. Estruturação das páginas em HTML
  3. Criação do arquivo CSS para estilização
  4. Adição de conteúdo: textos, imagens e tabelas
  5. Revisão e padronização de todas as páginas
  6. Publicação no GitHub Pages

Tecnologias Utilizadas

  • HTML5 — estrutura das páginas
  • CSS3 — estilização e layout
  • GitHub Pages — hospedagem gratuita
  • Google Fonts — tipografia personalizada
